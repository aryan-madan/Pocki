# pocki

a non-custodial crypto wallet for the ethereum sepolia testnet. built with a focus on security, simplicity, and a fluid user experience. pocki gives you full control over your digital assets with a clean interface and smooth interactions.

## features

- generate or import wallets securely
- encrypted local storage for private keys
- send and receive eth and erc-20 tokens
- real-time balance and transaction history
- customizable wallet cards
- dark and light mode support

## tech stack

- react
- typescript
- ethers.js
- tailwind css
- gsap
- three.js

## running locally

1. install dependencies
   `npm install`

2. start development server
   `npm run dev`

3. build for production
   `npm run build`
